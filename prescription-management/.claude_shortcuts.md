# Claude Shortcuts

## Quick Commands

### `/docs` - Read all documentation
Read these files in order:
1. CLAUDE.md
2. ENTITY_RELATIONSHIP_DIAGRAM.md  
3. API_REFERENCE_GUIDE.md
4. PROJECT_ARCHITECTURE.md

### `/search` - Search existing code
Use before creating any new functions:
```bash
rg "function_name" --type py
rg "class ClassName" --type py
```

### `/api` - Check API endpoints
Verify endpoints exist:
```bash
rg "@router\.(get|post|put|delete)" backend/app/api/
```

### `/structure` - Check folder structure
Never create files outside this structure (see PROJECT_ARCHITECTURE.md)
