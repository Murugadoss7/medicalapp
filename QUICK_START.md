# Quick Start

## Start Application
```bash
./start-app.sh
```

## Stop Application
```bash
./stop-app.sh
```

## Access
- **Frontend:** http://localhost:5173
- **Backend:** http://localhost:8000
- **API Docs:** http://localhost:8000/api/v1/docs

## Logs
```bash
tail -f /tmp/backend.log
tail -f /tmp/frontend.log
```
